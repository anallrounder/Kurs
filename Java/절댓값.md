**절댓값 구할 때 더 간결하게 쓸 수 있다.**

```java
//작성 코드
	int num = -10;
	if(num > 0)
		System.out.println(num); //쓸 필요 없다.
	else if(num < 0)
		System.out.println(-num);
	else 
		System.out.println(0); //쓸 필요 없다.

//개선된 코드 
	int num = -10;
	**if(num < 0)
		num = -num;**
				System.out.println("절댓값은: " + num);
```
